Path

There are 5 line commands for node paths:

  • "Move to" or M x y or m dx dy, which takes two parameters, coordinate x and y, and appears at the beginning of paths to specify where the drawing should start.
  • "Line to" or L x y or l dx dy, which takes two parameters, coordinate x and y, draws a line from the current position to a new position.
  • H and V are abbreviated forms for drawing a line, which take only one parameter as they only go in one direction:
    • H x (or h dx) for horizontal
    • V y (or v dy) for vertical
  • "Close Path" or Z draws a straight line from the current position back to the first point of the path. It is often placed at the end of a path node, although not always

Filters

SVG filters are defined within a <defs> element. The <defs> element is short for definitions and contains definition of special elements (such as filters).

The <filter> element is used to define an SVG filter. The <filter> element has a required id attribute which identifies the filter. The graphic then points to the filter to use.

The <feOffset> element is used to create drop shadow effects. The idea is to take an SVG graphic (image or element) and move it a little bit in the xy plane.
